Analizador de I/O Cost

Showplan

Instrucciones

  • Antes de la ejecución de cualquier proceso ejecutar la instrucción: 
    set showplan on
  • También debera de estar activado el panel «Show Log», si no esta activado o no se ve, utilizar Alt+2
  • Copiar todo lo mostrado en el «Show Log» seleccionando con click derecho y select all
  • Guardar el texto copiado en un archivo «txt», ese archivo es el que se debe de cargar.

Glosario de términos.

I/O cost:

In/Out (Entrada/Salida) Se refiere a los datos desde o hacía los discos y/u otros periféricos, es probablemente de los recursos más lentos. El «Costo Estimado de I/O» se encarga de recopilar los costes por parte de diferentes procesos que se aquí mismo se especificarán.

Table Scan:

El número total de páginas de datos en la tabla; si el proceso indica esta etiqueta significa que recorrerá toda la tabla para encontrar el elemento requerido en el proceso.

Clustered Index Scan:

El número de niveles en el índice más el número de páginas de datos a escanear
