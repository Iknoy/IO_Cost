  • Antes de la ejecución de cualquier proceso ejecutar la instrucción: set showplan on
  • Copiar lo mostrado en el «Show Log», si no esta activado o no se ve, utilizar Alt+2
  • Guardar el texto copiado en un archivo «txt», ese archivo es el que se debe de cargar.

¿Qué significa I/O cost? (El Costo Estimado de I/O o Costo Estimado de CPU)

El analizador de consultas crea muchos planes de ejecución, calcula el costo de cada uno (cuántos recursos de I/O y procesador son necesarios para completar la operación) y determina el más eficiente.
